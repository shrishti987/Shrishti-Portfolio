BreastScan AI

A machine learning-powered web application that predicts whether a breast tumor is benign or malignant using clinical features. Built to make cancer risk prediction more accessible and interpretable through an interactive UI.

Developed using Python and Streamlit, the system uses a soft-voting ensemble model combining Random Forest, Gradient Boosting, and Logistic Regression, achieving ~97–98% accuracy on the Wisconsin Breast Cancer dataset.

Beyond prediction, the application focuses on explainability and usability — users can visualize risk probability, understand feature contributions, upload CSV files for batch predictions, generate PDF reports, and track prediction history.

Travelia – Full-Stack Travel Booking Platform

Travelia is a full-stack travel booking platform inspired by Airbnb where users can explore listings, book stays, and manage trips. Built using Node.js, Express, MongoDB, and EJS with real-world usability in mind.

Key features include authentication, CRUD listings with image upload, dynamic booking system, wishlist, reviews, category-based filtering, a recommendation system, and fraud detection logic.

Architecture:

  • Frontend: EJS + Bootstrap
  • Backend: Express (routes & business logic)
  • Database: MongoDB (Mongoose)
  • Authentication: Passport.js (session-based)
  • File Upload: Multer + Cloudinary

Flow: User → Request → Express Routes → Business Logic → MongoDB → Response → EJS Render

Node.js Express.js MongoDB EJS + Bootstrap Passport.js Listings CRUD Booking System Recommendation System Fraud Detection Wishlist & Reviews

Unique Edge: Intelligent booking validations, recommendation logic, and fraud detection make this more than a basic CRUD application — it simulates real production-level travel platforms.
